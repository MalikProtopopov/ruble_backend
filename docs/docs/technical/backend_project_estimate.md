# Оценка бэкенда «По Рублю» — коммерческий проект

---

## 1. Общее описание продукта

Платформа для благотворительности с рекуррентными подписками, разовыми пожертвованиями, геймификацией (стрики, достижения), благодарностями от фондов и полноценной админ-панелью.

**Стек:** Python 3.11, FastAPI (async), PostgreSQL 16, SQLAlchemy 2 (async), Redis, S3/MinIO, YooKassa, Firebase Cloud Messaging, Taskiq (фоновые задачи).

- 99 API-эндпоинтов (публичные + админские)

## 6. Аргументы для продажи как продукт

1. **Готовый к продакшену** — задеплоен, работает на сервере, обрабатывает реальные платежи
2. **99 API-эндпоинтов** — полный CRUD для всех сущностей, не заглушки
3. **Платежи работают** — YooKassa интегрирована с вебхуками, рекуррентными списаниями, retry-логикой
4. **Покрыт тестами** — 28 тестовых файлов: интеграционные, API, unit
5. **Масштабируемая архитектура** — async, фоновые задачи, cursor-пагинация, атомарные операции
6. **Безопасность** — JWT RS256, GDPR (удаление аккаунта), валидация IP вебхуков, DOMPurify
7. **DevOps готов** — Docker Compose, nginx, SSL, deploy-скрипты, health checks
8. **Документация** — API guide (2000+ строк), технические доки по каждому модулю
